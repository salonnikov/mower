## НАХОДКИ ИЗ ВСКРЫТИЯ (2026-06) — OEM = «SNK» (вероятно Sunseeker)

Две платы + общий UART. Шелкскрин ПОДПИСАН — реверс лёгкий.

**Мейнборд `SNK_MAINBOARD_CP_V11` (PN 80102372-01):**
- Главный MCU: ARM Cortex-M, по виду **MM32 / GD32** (клон STM32), LQFP.
- **SWD/JTAG-хедер (обратная сторона):** `3V3 / DIO / CLK / JTDO / RES / GND`.
- Разъёмы: `J9 = LEFT / RIGHT (+GND)` моторы колёс; `J8 = GND / START / OK / +5V / ON`;
  `J7 = GND…`; сигналы `BLADE_1` (нож), `B_5V`, `B_AGND` (измерение тока АКБ), `STOP`.
- USB-порт = обновление прошивки с флешки FAT32 (на экране «USb»). НЕ канал управления.
- Питание от сборки 5S 18650 (~20 В).

**Плата дисплея `SNK_DISPLAY_CP_V11` (PN 80102373-01) = по сути ESP32-модуль:**
- **`ESP32-WROOM-32UE`** (dual-core, Wi-Fi+BT, внешняя антенна U.FL; FCC 2AC7Z-ESPWROOM32UE).
  Это ОН рулит дисплеем/кнопками/пищалкой/Wi-Fi и говорит с мейнбордом по UART.
- LED-драйверы 7-сегмента (SOIC), пищалка BUZ1, кнопки ON/START/HOME/OK.
- **Хедер прошивки на краю: `3V3 / T / R / GND / GND / P`** (T/R=UART0, **P=IO0/boot**) →
  можно снять дамп и перешить ESP32 на месте.
- Свой SWD-хедер тоже есть.
- Белый шлейф `J2` к мейнборду — питание (5V/GND) + **межплатный UART** (телеметрия+команды).

**ПЛАН (уточнён):** B — захватить штатный ESP32:
  0) `esptool read_flash` штатной прошивки (бэкап + в ней зашит протокол мейнборда);
  1) снифить межплатный UART для подтверждения;
  2) перешить ESP32 своим кодом → Wi-Fi/MQTT + управление по родному UART, дисплей/кнопки сохранить.

**Датчики:** одометрия — Холлы `KCD KD-002`, подписаны `LEFT`/`RIGHT` (залиты компаундом).

**ПЛАН (пути к цели):**
- A. Снифить межплатный UART (шлейф мейнборд↔дисплей) — родной протокол. ← старт.
- B. Перепрошить штатный ESP дисплея своим кодом (готовый Wi-Fi + UART к мозгу).
- C. SWD-дамп главного MCU через хедер.
- D. Прямая пересадка: рулить J9 LEFT/RIGHT + BLADE, читать Холлы/периметр.
