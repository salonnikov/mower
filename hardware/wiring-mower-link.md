# Подключение нашей ESP32 (mower-link) к плате дисплея косилки (J1)

Наша плата: ESP32 DOIT DevKit, 30 pin, USB снизу.
Все 4 провода — на ПРАВЫЙ ряд (D5, TX2, RX2 подряд + GND ниже).

```
                 ESP32 DOIT DevKit (30 pin)          J1 платы дисплея косилки
                    ┌──────────────────┐             ┌──────────────────────┐
              EN ──►┤                  ├──◄ D23      │  3V3  ── НЕ подключать │
          VP(36) ──►┤                  ├──◄ D22      │   T   ──► ESP RX2/16   │
          VN(39) ──►┤     ESP32        ├──◄ TX0      │   R   ──► ESP TX2/17   │
             D34 ──►┤    WROOM-32      ├──◄ RX0      │  GND  ──► ESP GND      │
             D35 ──►┤                  ├──◄ D21      │  GND  ── (любой GND)   │
             D32 ──►┤                  ├──◄ D19      │   P   ──► ESP D5       │
             D33 ──►┤                  ├──◄ D18      └──────────────────────┘
             D25 ──►┤                  ├──◄ D5     ◄═══════ J1 "P"  (IO0, дамп)
             D26 ──►┤                  ├──◄ TX2/17 ◄═══════ J1 "R"
             D27 ──►┤                  ├──◄ RX2/16 ◄═══════ J1 "T"
             D14 ──►┤                  ├──◄ D4
             D12 ──►┤                  ├──◄ D2
             D13 ──►┤                  ├──◄ D15
             GND ──►┤                  ├──◄ GND    ◄═══════ J1 "GND" (общая земля!)
             VIN ──►┤                  ├──◄ 3V3
                    └────────┬──────┬──┘
                          ═══╡ USB  ╞═══  ◄── питание ESP: повербанк/комп
                             └──────┘
```

| J1 косилки | → ESP32 | зачем |
|---|---|---|
| **T**   | **RX2 / GPIO16** | дисплей→мейнборд (и RX для дампа) |
| **R**   | **TX2 / GPIO17** | мейнборд→дисплей (и TX для дампа) |
| **P**   | **D5 / GPIO5**   | вход в загрузчик (только дамп) |
| **GND** | **GND**          | общая земля — обязательно |

ВНИМАНИЕ: J1 `3V3` НЕ подключать (ESP питается своим USB/повербанком).
Прошивка: mower-link (режимы SNIFF/BRIDGE). Пины заданы в firmware/mower-link/src/main.cpp.
