# chip2 — классификатор in/out (шаблон кодированного пакета)

Вывод `pdg` — доверяем. `⚑` = интерпретация (`ⓗ`).

## Классификатор последовательности сегментов — `fcn.080159a0`

`⚑ ⓗ` Сверяет накопленную последовательность сегментов (списки в объекте `in_r0`) с шаблоном кодированного пакета провода. Возвращает **код 0..4** — по коду доказано, семантика (какой код = «внутри»/«снаружи») — `ⓗ` из контекста.

**Доказательства (`✔`) из кода:**
- Ранний выход `return 1` при вырожденном входе (мало сегментов: `count < 4` или `count < n`).
- Допуск позиционного совпадения: `uStack_58 = coil[0x78]*4 - 100` (`✔` — масштабируемый порог от опорного периода).
- Матчер соседних сегментов `fcn.080179f8(..., 0x32, ...)` — допуск **`0x32` = 50** (`✔`).
- Порог «почти равно» **`< 10`** для схождения периодов (`✔`: `if (uVar6 < 10) { iStack_40=0; ... }`).
- Требует серий совпадений: `1 < uStack_6c` (≥2 в первой фазе) и `3 < uStack_44` (≥4 во второй) (`✔`).
- Финальная классификация по чистоте серии:
  - все сегменты `[+1]==0` (`uVar6 == uVar4`) → **`return 3`** (`✔`).
  - все сегменты `[+1]==1` (`uVar6 == uVar1`) → **`return 4`** (`✔`).
  - совпадение по первой фазе → **`return 0`**; иначе **`return 2`** (`✔`).

`⚑ ⓗ` Итоговый код состояния катушки формирует **не сам классификатор**, а вызывающий `follow_border_check_wave` (`fcn.080164bc`), мапя `return` классификатора в 0..8 (см. 04). Соответствие «0=нет/1=внутри/2=снаружи/…» — из контекстного дока, требует независимой проверки на живом сигнале.

uint fcn.080159a0(...)   // ⓗ classify_packet; r0=segset, r1=n, r2=?, r3=out
{
    if (bad_args || segset.count < 4 || segset.count < n || n==0) return 1;   // ✔
    if (segset.count == n) return 2;
    uint tol = coil[0x78]*4 - 100;                 // ✔ масштаб. допуск

    // --- фаза 1: цепочки сегментов полярности 0 ---
    for (each seg with [+1]==0) {
        run = 0;
        for (следующие seg' с [+1]==0, позиционно рядом) {
            if (fcn.080179f8(seg', seg, r2, 0x32, ...) == 0) {   // ✔ match, tol 50
                seg = seg'; run++;
                if (run > 1) { *out = seg0; matched = 1; }        // ✔ ≥2
            }
        }
        if (matched) return 0;                                    // ✔
    }
    // --- фаза 2: цепочки сегментов полярности 1 ---
    for (each seg with [+1]==1) {
        run = 0;
        for (следующие seg' с [+1]==1, рядом) {
            if (fcn.080179f8(...,coil[0x78],0x32,...) == 0) {
                seg = seg'; run++;
                if (run > 3) {                                    // ✔ ≥4
                    // проверить схождение периода к опорному coil[0x78] c допуском <10
                    if (|period - ref| < 10) { best = seg; matched = 1; }  // ✔
                }
            }
        }
        // выбрать лучший кандидат в *out
    }
    if (!matched) return 0;

    // --- чистота серии по полярности ---
    count0 = count of segs with [+1]==0;
    count1 = count of segs with [+1]==1;
    if (total == count0) { *out = extreme_by_peak(...); return 3; }   // ✔ все «0»
    if (total == count1) { *out = extreme_by_peak(...); return 4; }   // ✔ все «1»
    return 2;
}
