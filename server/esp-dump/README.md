# Автоматический дамп прошивки ESP32 косилки

Через мост `mower-link` (режим BRIDGE) по Wi-Fi. Без USB, без ручных команд esptool.
`read_flash` — ТОЛЬКО чтение, косилку не ломает.

## Шаги
1. Прошей `mower-link` на нашу ESP32, подключи к J1 косилки:
   `T→GPIO16, R→GPIO17, P→GPIO5, GND→GND` (питание нашей ESP — свой USB/повербанк).
2. На странице `mower-link` (http://mower-link.local/) нажми **BRIDGE (дамп)**.
3. На домашнем ноуте:
   ```
   cd server/esp-dump
   BRIDGE_IP=<ip-mower-link> docker compose run --rm dump
   ```
4. Когда скрипт попросит — **передёрни питание косилки** (её ESP уйдёт в загрузчик).
5. Скрипт сам поймает ESP, прочитает 4 МБ, сохранит `out/mower-esp-<дата>.bin`
   и скажет **вердикт**: нормальная прошивка (есть протокол) или зашифрована.
6. После — верни `mower-link` в **SNIFF** и передёрни питание косилки (вернётся к норме).

## Если «возможно зашифрован»
Тогда дамп бесполезен — переключаемся на **сниф протокола** (mower-link SNIFF + логгер
в `server/uart-logger`), разбираем команды/телеметрию по живому потоку.
