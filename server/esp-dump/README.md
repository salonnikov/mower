# Снятие дампа прошивки ESP32 косилки через мост (mower-link, режим BRIDGE)

Запускать на домашнем ноуте (он же 192.168.10.66). USB не нужен — всё по Wi-Fi.

## Порядок
1. На странице `mower-link` нажми **BRIDGE (дамп)** — ESP начнёт держать P(IO0) в GND.
2. **Передёрни питание косилки** — её ESP32 уйдёт в режим загрузчика (UART download).
3. Запусти esptool в контейнере (читает флеш по сети через мост, ТОЛЬКО чтение):

```
mkdir -p out
docker run --rm -v "$PWD/out:/out" python:3.12-slim bash -lc "\
  pip install -q esptool && \
  esptool --before no_reset --after no_reset --chip esp32 \
          --port 'socket://<IP-mower-link>:3333' \
          read_flash 0 0x400000 /out/mower-esp.bin"
```
(подставь IP моста — он виден на странице mower-link.)

Результат: `out/mower-esp.bin` — бэкап прошивки + внутри протокол общения с мейнбордом.

## Если не вышло
- «Failed to connect / no sync» → косилкина ESP не в загрузчике: убедись, что BRIDGE включён ДО передёргивания питания, и передёрни ещё раз.
- Дамп — мусор/одни 0xFF/ошибка чтения → флеш зашифрован (flash encryption). Тогда дамп бесполезен, идём по протоколу (сниф).
- read_flash НИЧЕГО не пишет — косилку не сломает. После дампа верни режим SNIFF и передёрни питание (P отпустится, ESP загрузится нормально).
